----------------------------------------------------------------
                     Указания за стартиране
----------------------------------------------------------------

1. ПРЕДВАРИТЕЛНИ ИЗИСКВАНИЯ:
   - Трябва да имате инсталиран .NET 8.0 SDK.
   - Трябва да имате инсталиран SQL Server (LocalDB по подразбиране).

2. ПОДГОТОВКА НА БАЗАТА ДАННИ:
   - Отворете SQL Server Management Studio (SSMS) или Azure Data Studio.
   - Изпълнете файла 'LearnConnect_Full_Script.sql', за да създадете базата данни и да заредите началните данни.

3. СТАРТИРАНЕ НА ПРИЛОЖЕНИЕТО:
   - Най-лесният начин е да стартирате файла 'START.bat' от основната директория.
   - Скриптът автоматично ще компилира проекта и ще стартира сървъра.
   - Браузърът ви ще се отвори автоматично на адрес: https://localhost:5001

4. РЪЧНО СТАРТИРАНЕ (през терминал):
   - Отворете терминал в папка 'LearnConnect.API'.
   - Изпълнете командата: dotnet run
